kuBot configuration utility is live!

This website was designed to help you configure kuBot. kuBot is a discord bot used to report activity on discoverygc server.

Bots features

  • Watch targets definition using tags (factions) and regions (systems).
  • Scan command : display activity depending on the watch lists configured - factions, regions and players.
  • Activity warnings.
  • Embed messages utilities.

How to add this bot to my guild?

Easy! You can use this link: Bot invite.

Create a temporary "admin" text channel to configure the bot.

You can then use the !config command in that newly created "admin" channel to generate credentials for this website.
