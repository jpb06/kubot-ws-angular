About the kubot project

Muh nodejs, muh typescript.

Project history

This all started with a desire to make a little bot for KNF discord and practice some javascript along the way. Little by little, the project made me study various technologies and it was good fun all along!

A little word about architecture...

The project is split into logical blocks:

  • The discord bot, written in Typescript and serving as the main app of the project.
  • The Angular 6 website, made to provide an easy tool for configuring the bot.
  • The frontend API, used by the website to do what it does best.
  • The Data Access Layer, provided as a NPM package and written in Typescript as well.

All these fine little things run in a self hosted environment relying on Debian, nginx and mongodb. The challenge was to put this all together and secure on a 100% home made config.

Security concerns

The implementation allowed me to experiment some pretty things like json web tokens (JWT) and HTTPS using let's encrypt.
